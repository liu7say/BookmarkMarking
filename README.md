# Bookmark Manager 书签管理器 (Chrome 侧边栏插件版)

基于 **Vue 3 + Vite + Dexie.js (IndexedDB)** 构建的高级浏览器书签管理与打标工具，采用 **Fluent 2** 设计语言，现已深度集成并适配为 **Chrome Extension 侧边栏 (Side Panel)** 插件。

---

## 🚀 核心功能特性

### 1. Chrome 插件特有（侧边栏原生集成）
*   **侧边栏常驻显示 (Side Panel)**：点击浏览器工具栏的图标，即可在右侧/左侧拉出独立侧边栏。一边浏览网页，一边进行书签整理，完美的双线程交互。
*   **键盘快捷键 (Toggle Key)**：支持在浏览器中随时按下 **`Ctrl + Shift + X`** (Mac 上为 `Cmd + Shift + X`) 快速呼出/隐藏侧边栏，极速提效。
*   **实时双向数据同步**：原生挂载 Chrome 书签变更监听事件 (`onCreated`, `onRemoved`, `onChanged`, `onMoved`)。在浏览器上做的任何书签整理改动，侧边栏都会瞬间响应，零延迟同步到本地 IndexedDB 库中。
*   **⚡ 快捷保存当前活动页**：智能捕捉当前浏览的活动标签页。一键点击即可实现快速收录，并调用 AI 自动推荐标签。
*   **特权级高速 Favicon 抓取**：利用扩展特权协议，直接且瞬时地从 Chrome 本地缓存中抓取 Favicon 图标，网络延迟为 0，彻底根除跨域 (CORS) 限制。

### 2. 本地化整理与自定义标签 (Tagging)
*   **一键同步浏览器书签**：支持将 Chrome 真实的书签目录一键递归扁平化载入本地，并与现有数据库进行智能去重（按 URL 比对），**绝对不破坏和覆盖现有的自定义打标数据**。
*   **本地书签导入**：保留标准的 HTML 书签文件拖拽/点击上传（完美向下兼容）。
*   **树形层级浏览**：以嵌套文件夹卡片的形式，支持平滑平移（150-300ms Fluent 2 曲线）地展开与折叠。
*   **自定义打标**：支持为任意书签打上自定义 Tag，支持多值联合查询。
*   **AI 智能辅助工具**：
    *   **网站图标抓取**：对缺少图标的旧书签进行后台自动批量抓取。
    *   **AI 自动打标**：结合网页上下文推荐多维标签。
    *   **AI 网页标题提取**：自动将直接以 URL 作标题的书签还原为规范的网页名称。
    *   **URL 批量解码**：智能解码被百分号编码的各种中文 URL 路径。
*   **开发者 JSON 控制台**：终端风格的底层数据控制面板，支持对 IndexedDB 原始数据的一键复制和结构检查。

### 3. Fluent 2 UI & 极窄自适应
*   **视觉艺术**： mica 柔和渐变背景、高透明度半透“亚克力”玻璃卡片、高光立体描边。
*   **极窄侧边栏自适应 (`< 600px`)**：为 300px-400px 侧边栏极限空间做深度微调：隐藏冗余 Logo、自动折叠导航菜单、限制 URL 最大宽度以支持省略号、在非 Hover 触控设备上使操作按钮默认可见。

---

## 🛠️ 技术栈

*   **前端核心**：Vue 3 (Composition API, `<script setup lang="ts">`) + TypeScript
*   **状态管理**：Pinia
*   **本地数据库**：Dexie.js (IndexedDB 包装，大容量、纯本地安全、免后端)
*   **构建工具**：Vite (资源相对路径打包适配 `'./'`)
*   **UI 框架**：自定义 Fluent 2 CSS 系统
*   **图标库**：Lucide Icons

## 📦 开发与构建

### 1. 运行本地开发环境
```sh
npm install
npm run dev
```

### 2. 构建打包为 Chrome 插件
编译打包输出在 `dist/` 目录：
```sh
npm run build
```

---

## 🔌 插件加载指引

1. 打开 Chrome 浏览器，访问地址：**`chrome://extensions/`**。
2. 开启右上角的 **“开发者模式” (Developer mode)**。
3. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
4. 在弹出的文件选择器中，选择本项目编译生成的 **`dist`** 文件夹。
5. 在浏览器右上角的扩展栏中，将 **“书签整理大师”** 钉选 (Pin) 到工具栏。
6. 点击图标，即可直接在侧边栏中开启高端、流畅的书签整理工作！
